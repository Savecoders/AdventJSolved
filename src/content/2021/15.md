## Mi Solución

No olvides que esta es mi solucion, recuerda primero intentarlo tú mismo.

```js
const checkSledJump = heights =>
  heights.reduce(
    (acc, item, i) => (heights[i - 1] < item || heights[i - 1] === item ? acc + 1 : acc - 1),
    0,
  ) < 0;
```
