## Mi Solución

```js
const canCarry = (capacity, trip) => {
  let Baggage = {};
  trip.forEach(([gifts, pickUp, delivery]) => {
    capacity -= gifts - (pickUp in Baggage ? Baggage[pickUp] : 0);
    Baggage[delivery] = gifts;
  });
  return capacity >= 0;
};
```
